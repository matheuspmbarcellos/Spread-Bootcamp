# Funções em javascript


### Declarando uma função:

    function soma(a, b) {
        return a + b
    }

### Chamando uma função:

    function soma(a, b) {
        return a + b
    }

    soma(3, 5)    <----

### Mostrando uma função no console:

    function soma(a, b) {
        return a + b
    }

    console.log(soma(3, 5))    <----
